# Frontend for Waco Node.js Technical Test

This project provides the frontend interface for managing users and their favorite Pok√©mon.

## Requirements

- Node.js (v16 or higher recommended)

## Installation

1. Clone the repository.
2. Navigate to the project directory.
3. Run `npm install` to install the dependencies.

## Development

You can start the development server using:

```bash
npm run dev
```

This will start the development server, and you can access the application at `http://localhost:3000`.

## Building the Project

You can build the project for production using:

```bash
npm run build
```

This will create a `dist` directory with the compiled assets.

## Previewing the Build

You can preview the production build using:

```bash
npm run preview
```

## Linting

You can run the linter to check for code quality issues using:

```bash
npm run lint
```
